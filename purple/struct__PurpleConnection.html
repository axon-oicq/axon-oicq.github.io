◆ wants_to_die

gboolean _PurpleConnection::wants_to_die

Wants to Die state. This is set when the user chooses to log out, or when the protocol is disconnected and should not be automatically reconnected (incorrect password, etc.). prpls should rely on purple_connection_error_reason() to set this for them rather than setting it themselves.
