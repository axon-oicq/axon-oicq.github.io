Creates a node from a XML File. Calling this on the root node of an XML document will parse the entire document into a tree of nodes, and return the xmlnode of the root.

Parameters
dirThe directory where the file is located
filenameThe filename
descriptionA description of the file being parsed. Displayed to the user if the file cannot be read.

Returns
